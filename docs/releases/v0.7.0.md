# Sprint 7B — Frontend Foundation

## Objetivo

Criar a fundação técnica e visual do frontend sem implementar a landing completa.

## Arquitetura

Frontend único em Next.js App Router para landing pública, autenticação e dashboard futuro. Spring Boot permanece como único backend e fonte de regras de negócio.

## ADR

Criado `docs/adr/ADR-006-frontend-rendering-strategy.md`.

## Endpoint público

Contrato público consolidado:

- `GET /public/sites/{companySlug}`
- `GET /public/sites/{companySlug}/services`
- `GET /public/sites/{companySlug}/gallery`

O endpoint de site expõe apenas dados públicos de empresa, localização pública e branding.

## Estrutura frontend

Criado scaffold em `frontend/src` com App Router, componentes de layout, providers, cliente REST, env validation, estilos, tipos e testes.

## Design tokens

Tokens semânticos implementados em CSS custom properties. Componentes usam tokens, não valores específicos de tenant.

## SEO

Metadata base, title template, Open Graph, viewport, robots e sitemap criados.

## Acessibilidade

Incluídos landmarks semânticos, skip link, foco visível, `lang="pt-PT"`, headings hierárquicos, áreas interativas adequadas e estados discretos.

## Testes

Testes básicos adicionados para env, API client, branding, header, footer e fallback de API.

## CI

Workflow frontend separado com install via lockfile, lint, typecheck, test e build. Workflow backend preservado.

## Riscos

- URLs reais de logo dependem da estratégia final de storage/CDN.
- Tenant por domínio ainda não foi implementado.
- Conteúdo real da landing continua dependente da Sprint 7C.

## Critérios de aceite

- ADR-006 criado.
- Endpoints públicos por slug implementados.
- Frontend inicia e renderiza `/` com header, placeholder técnico e footer.
- Tokens, branding seguro, SEO, estados globais, testes e CI configurados.
- Nenhuma secção real da landing implementada.

## Próxima Sprint

Sprint 7C — implementar a landing pública com hero, serviços, galeria, processo, sobre, área geográfica e contacto, consumindo os contratos públicos definidos.
