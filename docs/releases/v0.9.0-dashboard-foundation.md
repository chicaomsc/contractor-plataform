# Release v0.9.0 — Dashboard Foundation

**Data:** 2026-07-16  
**Sprint:** 8A — Dashboard Foundation

---

## Objectivo

Iniciar a área autenticada do Contractor Platform com a fundação visual e técnica do dashboard administrativo, sem criar funcionalidades novas de negócio, sem alterar arquitectura, sem alterar backend e sem criar endpoints.

---

## Implementado

### Área autenticada

- Login frontend consumindo `POST /auth/login`
- Sessão client-side com access token e refresh token retornados pelo backend
- Validação da sessão com `GET /auth/me`
- Logout com limpeza de sessão local
- Middleware para proteger `/dashboard`
- Guard client-side para validar sessão real antes de renderizar a área autenticada

### Dashboard Layout

- Sidebar desktop
- Drawer mobile
- Header autenticado
- Breadcrumb
- Área principal responsiva
- User menu com nome, role e logout
- Navegação para Início, Empresa, Branding e Settings

### Dashboard Home

Mostra apenas dados existentes no backend:
- Empresa
- Status do site
- Estado do branding
- Moeda configurada
- Quantidade de serviços via `GET /services`
- Quantidade de imagens via `GET /gallery`
- Última atualização baseada em `updatedAt` de serviços e galeria

Não foram criados gráficos, métricas fictícias ou dashboards financeiros/operacionais.

### Company

Tela completa de edição consumindo:
- `GET /company/me`
- `PUT /company/me`

Campos:
- Nome
- Nome comercial
- Email
- Telefone
- WhatsApp
- Website
- NIF / tax number
- País
- Morada completa

### Branding

Tela completa de edição consumindo:
- `GET /branding/me`
- `PUT /branding/me`

Campos:
- Cor primária
- Cor secundária
- Cor de acento
- Tagline
- Sobre
- Texto de rodapé
- Prefixo de orçamento
- Assinatura

Inclui preview em tempo real usando os valores do formulário e o logo já existente no backend.

### Settings

Tela completa de edição consumindo:
- `GET /settings/me`
- `PUT /settings/me`

Campos:
- Moeda padrão
- Taxa padrão
- Validade de orçamento
- Texto padrão de rodapé do orçamento
- Locale
- Timezone
- Formato de data
- Formato numérico

## Integração com Backend

Endpoints utilizados:
- `POST /auth/login`
- `GET /auth/me`
- `GET /company/me`
- `PUT /company/me`
- `GET /branding/me`
- `PUT /branding/me`
- `GET /settings/me`
- `PUT /settings/me`
- `GET /services`
- `GET /gallery`

`/services` e `/gallery` são usados somente para contagens e última atualização na Home. Não foi implementada gestão de serviços ou galeria.
